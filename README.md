# Agent Playground

A personal experimentation workspace for building and testing different AI agents.

## Overview

This project is a playground for creating and testing various AI agents for fun and personal use. Each agent experiment is organized in its own directory with its own dependencies and configuration.

## Current Agents

- **simple-agent** - A minimal pi-mono experiment featuring an agent with a tool that sums numeric literals in the latest user message. Uses OpenRouter via the @mariozechner/pi-agent-core and @mariozechner/pi-ai packages.
- **sample-openclaw-agent** - A PHP CLI daemon that mirrors TypeScript Slack + Anthropic agent behavior. Native PHP implementation based on Hugo Dutka's OpenClaw design with Slack Socket Mode integration and Anthropic API.

## Getting Started

Each agent directory is self-contained. Navigate to the specific agent directory and follow its README for setup and usage instructions.

## Purpose

This repository serves as a personal testing ground for:
- Experimenting with different AI agent frameworks
- Learning agent architecture and tool design
- Testing LLM integrations and capabilities
- Building custom tools and workflows
