## Description
Add a new extraction mode called "html-templates" that extracts the PDF layout, contents, and formatting by generating a visually faithful HTML replica of the PDF.

## Implementation Details
- [ ] Add `HTML_TEMPLATES` to `ExtractionMode` enum in schemas.py
- [ ] Create HTML template system for PDF layout replication
- [ ] Generate structured output schema for HTML extraction:
  - Page layout structure (columns, sections, etc.)
  - Typography (fonts, sizes, weights)
  - Images and positioning
  - Tables and structured data
  - Colors and backgrounds
- [ ] Add HTML rendering logic in analyzer.py
- [ ] Consider using CSS Grid/Flexbox for layout fidelity
- [ ] Support embedded images (base64 or linked)
- [ ] Add CLI support: `--mode html-templates`

## Files Affected
- `src/pdf_extractor_analyzer/schemas.py` - Add mode to enum
- `src/pdf_extractor_analyzer/analyzer.py` - Add HTML extraction logic
- `src/pdf_extractor_analyzer/pipeline.py` - Handle HTML mode
- `src/pdf_extractor_analyzer/cli.py` - Add CLI flag
