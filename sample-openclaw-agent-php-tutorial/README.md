# sample-openclaw-agent-php Tutorial

Interactive codebase tutorial for **sample-openclaw-agent-php**.

## Features

- Vite + React + TypeScript
- Sidebar chapter navigation
- Progress + quiz score tracking (localStorage)
- Syntax-highlighted code blocks with `prism-react-renderer` (`vsLight`)
- SVG diagrams for architecture and flow
- `tutorial-manifest.json` with source file SHA-256 hashes for drift detection

## Chapters

1. Architecture Overview
2. Key Modules and Their Purposes
3. Data Flow: Slack Event to Agent Reply
4. TypeScript Patterns in the Reference Port
5. Entry Points and Configuration

## Run

```bash
npm install
npm run dev
```

Build production bundle:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```
